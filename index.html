⟡ SUBSET CONSTRUCTION ALGORITHM
1

INITIALIZE

Compute ε-closure({start}). This set of NFA states becomes the first DFA state. Add it to the work queue.

2

EXPAND (loop until queue empty)

For each unprocessed DFA state S and each symbol σ:
a) move(S,σ) = NFA states reachable on σ
b) ε-closure(move(S,σ)) = target DFA state
c) Record transition S →[σ]→ target
d) If target is new, enqueue it

3

ACCEPT STATES

A DFA state is accepting if it contains any NFA accepting state in its subset. The DFA accepts exactly the same language as the NFA.

4

DEAD STATE {∅}

If no NFA states are reachable the result is {} — the dead/trap state. All transitions from it loop back. No string is accepted from there.

ε

EPSILON CLOSURE

ε-closure(S) is all NFA states reachable from S using only ε-transitions (no input consumed). Computed by BFS/DFS from every state in S.
